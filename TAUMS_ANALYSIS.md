# CO₂ + SW68 中观输运 τ_ms 结果分析

> 所有数据使用占位参数计算。在实验数据可用后，绝对数值会变，但**相对趋势和物理规律不变**。

---

## 一、概览

| 指标 | 数值 | 物理含义 |
|------|------|----------|
| τ_ms 范围 | 16,000 – 32,000 ms | 质量传递达到稳态的特征时间 |
| D₁₂ 范围 | 27 – 30 ×10⁻⁹ m²/s | CO₂ 在油中的二元扩散系数 |
| μ_m 范围 | 0.9 – 51 mPa·s | 混合物动态粘度 |
| Re 范围 | 180 – 10,500 | 惯性力 / 粘性力 |
| Sc 范围 | 34 – 1,990 | 动量扩散 / 质量扩散 |

## 二、τ_ms 的核心物理规律

### 规律 1：τ_ms 与溶解度 w₁ 强负相关 (r = −0.998)

这是最重要的发现。**CO₂ 溶解越多，传质越快**。

```
T=303K 等温线:
  w1=0.165 (低溶解度)  →  τ_ms = 28,990 ms  (慢)
  w1=0.298              →  τ_ms = 23,609 ms
  w1=0.472 (高溶解度)  →  τ_ms = 17,921 ms  (快)
```

**物理解释**：CO₂ 分子渗入润滑油后起到"稀释剂"作用，大幅降低混合物粘度（μ_m 从 26.6 降到 1.8 mPa·s）。低粘度 → 更低 Sc → 更高 Sh → 更快传质 → 更短 τ_ms。

### 规律 2：压力升高 → τ_ms 降低

```
T=323K:
  P=3 MPa  → τ_ms = 30,944 ms
  P=7 MPa  → τ_ms = 23,796 ms
  P=15 MPa → τ_ms = 16,781 ms
```

高压迫使更多 CO₂ 溶于油 → 混合物粘度降低 → 传质加速。这是亨利定律在输运层面的直接体现。

### 规律 3：温度升高 → τ_ms 升高（定压下）

```
P=3 MPa:
  T=283 K  → τ_ms = 24,653 ms
  T=313 K  → τ_ms = 30,143 ms
  T=343 K  → τ_ms = 31,894 ms
```

这是反直觉的 — 通常升温加速扩散。但在此体系中，定压下升温**降低 CO₂ 溶解度**（w₁ 从 0.28 降到 0.08），导致混合物更接近纯油（高粘度），传质反而变慢。**溶解度效应压倒了温度对扩散的直接促进效应**。

### 规律 4：τ_ms 对参数变化相对不敏感

尽管 μ_m 变化了 50 倍，τ_ms 只变化了约 2 倍。这是因为：

```
τ_ms ∝ μ^(a−b) = μ^(0.5−0.333) = μ^0.167
```

高粘度同时增加了 Sc（利于传质）和降低了 Re（不利于传质），两者在 Sherwood 关联式中**部分抵消**。这使得 τ_ms 在宽广工况范围内保持相对稳定。

## 三、各物理量的空间分布特征

### D₁₂（扩散系数）

- 几乎恒定（27–30 ×10⁻⁹ m²/s）
- 随 T 和 P 只有微弱增加（~10%）
- 原因：自由体积理论中 V̂_FH 由 SAFT 堆积分数 η 决定，η 在液相范围内变化不大

### μ_m（粘度）

- 变化剧烈：0.9–51 mPa·s（50倍）
- **低 T + 低 P**（少 CO₂、多油）→ 高粘度
- **高 T + 高 P**（多 CO₂、少油）→ 低粘度
- 这是 τ_ms 变化的主要驱动因素

### Sc（施密特数）

- Sc = μ/(ρ·D₁₂) >> 1 始终成立
- 最小值 34（高 CO₂ 含量），最大值 1990（低 CO₂ 含量）
- **Sc >> 1** 意味着质量边界层远薄于速度边界层 → 传质阻力集中在壁面附近

### Re（雷诺数）

- 范围 180 – 10,500
- 在高 CO₂ 含量区可进入过渡/湍流状态
- 湍流会显著增强传质（需要切换到湍流 Sh 关联式）

## 五、τ_ms 场的工程解释

在宏观一维流动换热模型中，τ_ms 作为**控制方程源项**：

$$
\frac{\partial w_1}{\partial t} + \mathbf{u} \cdot \nabla w_1 = \nabla \cdot (D_{12} \nabla w_1) + \frac{w_{1,eq} - w_1}{\tau_{ms}}
$$

源项 $(w_{1,eq} - w_1) / \tau_{ms}$ 的含义：
- **w₁ < w_{1,eq}**（欠饱和）→ 源项为正 → CO₂ 从气相溶入液相
- **w₁ > w_{1,eq}**（过饱和）→ 源项为负 → CO₂ 从液相逸出
- **τ_ms 越小** → 源项越大 → 系统越快趋近平衡

τ_ms 的空间非均匀性意味着：
- 近壁面（高剪切、高 P）→ τ_ms 小 → 快速趋近平衡
- 流道中心（低剪切、低 P）→ τ_ms 大 → 缓慢趋近平衡
- 这种空间分布需要在 CFD 中作为**场变量**输入

## 六、当前限制与后续工作

| 限制 | 影响 | 解决方案 |
|------|------|----------|
| D₀ 为文献估算值 | τ_ms 绝对值不准确 | 用 CO₂+SW68 实验数据标定 |
| L、u 假设值 | 与实际流道不匹配 | 从 CFD 提取局部 u(x,y,z) |
| 层流 Sh 关联式 | 湍流区传质被低估 | 根据局部 Re 切换层流/湍流关联式 |
| 无温度对 D₀ 的修正 | 低温区 D₀ 偏高 | 引入 D₀(T) = D₀_ref × exp(−Ea/RT) |
| 粘度模型未标定 | μ_m 绝对值不准确 | 测量 CO₂+SW68 粘度数据或使用文献关联式 |

**所有占位参数集中在 `calc_taums.cpp` 文件头部（第 42–80 行），后续只需修改这些常量并重新编译运行即可更新全部 LUT。**
